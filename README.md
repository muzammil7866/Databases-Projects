# Database Projects

An ongoing collection of database coursework, lab work, and applied projects covering the full spectrum of relational database systems — from conceptual design through to SQL programming and database-driven application development. New projects are added as the curriculum and personal exploration progresses.

## Areas Covered

**Database Design**  
Entity-relationship modelling, enhanced ERD, schema normalisation, and translating real-world requirements into structured relational schemas.

**SQL & Relational Databases**  
DDL, DML, joins, subqueries, views, constraints, and query optimisation across different RDBMS platforms.

**Procedural SQL**  
Stored procedures, functions, triggers, cursors, and control structures using PL/SQL and similar procedural extensions.

**Database-Backed Applications**  
Desktop and application-layer projects that integrate relational databases, covering role-based access, CRUD operations, and data-driven workflows.

**Lab Exercises & Solutions**  
Structured practical exercises spanning course curricula, with corresponding worked solutions.

## Tools & Technologies

Projects in this repository may use any combination of the following:

Oracle Database XE · MySQL · Microsoft Access · PL/SQL · ADO.NET · C# .NET · MS Visio · SQL\*Plus · SQL Developer · and others as projects evolve.
