## Topics Covered

### Database Design
- **ERD (Lab 4):** Entity-relationship diagrams for real-world case studies drawn in MS Visio
- **EERD (Lab 5):** Extended ERD with specialization, generalization, and aggregation; includes a separate assumptions document justifying design decisions

### SQL with Oracle Database XE
- **DDL (Labs 6, 9):** `CREATE`, `ALTER`, and `DROP` statements; table constraints and schema modification on Oracle
- **Oracle SQL (Lab 7):** `SELECT`, `INSERT`, `UPDATE`, `DELETE` on built-in EMP and DEPT tables
- **Joins (Lab 8):** `INNER JOIN`, `OUTER JOIN`, aggregate queries across the EMP/DEPT tables
- **Views (Lab 10):** Creating and querying filtered database views (e.g., `HighSalaries`)

### PL/SQL Programming
- **Variables and Control Structures (Lab 11):** `%TYPE`, `%ROWTYPE`, `IF-ELSE`, loops, incentive calculations
- **Stored Procedures and Functions (Lab 12):** Parameterized procedures, named functions, `DBMS_OUTPUT`
- **Triggers (Lab 13):** `BEFORE`/`AFTER` DML triggers, salary validation, audit logging
- **Cursors (Lab 15):** Explicit cursors, parameterized cursors, `CURSOR FOR` loops
