## Overview

This folder contains all lab-related material produced during CL2005 – Database Systems Lab. The course progresses from application-level database usage and conceptual design through to relational SQL and PL/SQL programming on Oracle Database Express Edition.

## Course Progression

| Phase | Labs | Topics |
|-------|------|--------|
| Application Development | 1 – 3 | C++, Microsoft Access, C# Windows Forms |
| Database Design | 4 – 5 | Entity Relationship Diagrams (ERD, EERD) using MS Visio |
| SQL & Oracle Database | 6 – 10 | DDL, DML, Joins, Views on Oracle XE and MySQL |
| PL/SQL Programming | 11 – 15 | Variables, Procedures, Functions, Triggers, Cursors |
