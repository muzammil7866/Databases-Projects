## Assignment Details

### Assignment 1 – Oracle Database Installation

**Topic:** Oracle Database Express Edition (XE) Setup  
**Objective:** Install and configure Oracle Database XE on a local Windows machine as the primary DBMS for all subsequent lab exercises.

**Key steps covered:**
- Downloading Oracle Database Express Edition from the official Oracle website
- Running the Windows installer with appropriate port and configuration settings
- Setting up the `SYSTEM` and `SYS` administrator passwords
- Configuring environment variables (`ORACLE_HOME`, `PATH`)
- Verifying the installation by connecting via SQL*Plus
- Running initial test queries to confirm a working Oracle environment

**Tools:** Oracle Database XE 21c · SQL*Plus · Windows OS
